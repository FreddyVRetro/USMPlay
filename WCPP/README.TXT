Here are more info to help you to build this example:

With PMODE/W
------------

 - Copy pmodew.exe in \wc

 - Change the path to WATCOM include directory in makefile  

 - Run the borland make.exe ot the watcom wmake.exe program

With DOS4G/W
------------
 
 - Change the library file name

 - Change the SYSTEM section to dos4g

 - Use /dUseDOS4GW_C switch instead of /dUsePMODEW_C

If you want to change the LIB files, make sure you are using TASM v4.0.
